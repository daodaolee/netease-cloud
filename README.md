# netease-cloud
Mac端网易云音乐没有云盘上传，自己就做了个electron程序

## 安装
```
npm install
```

### 运行
```
npm run serve
```

### 浏览器打包
```
npm run build
```
### Electron运行
```
npm run electron:serve
```

### Electron打包
```
npm run electron:build
```
